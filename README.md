# Record Bae
Welcome to Record Bae!

This a free platform for Record Stores to sell albums during the pandemic made using Ruby on Rails.

## Installation 

In terminal

```bash
bundle install
```
followed by;
```bash
rails s
```

## run
```
In a browser of your choice
```
Enter https://localhost:3000
```
